Mock Course Website - Text AJAX

Overview

This is a mock page for a course home page. This section would contain the static HTML and the section below would contain the dynamic material. The section below, in a div with an id of schedule should display an up-to-date schedule of upcoming events which will be retrieved from a text file called schedule.html. schedule.html should be available for download on your course website. Once downloaded, upload it to your webspace in the directory where you will be uploading this page. Add the necessary Javascript to this page to retrieve schedule.html using AJAX then display the retrieved file in the "schedule" div.
